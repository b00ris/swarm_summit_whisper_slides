Whisper usage

Whisperv5

Main goal: the dark. It`s impossible to say what user reads or whiter what messages.

  • ddos prevention(TTL, PoW)
  • Filters and topics

Whisperv6

Mail goal: the good tradeoff between keeping darkness and traffic reduction.

Whisperv6

  • Full and light nodes were introduced
  • Full node has `nil` or all bytes 1 Bloom filter
  • Light node has some different bloom filter
  • Bloom filter has many collisions per topic by design to not to share user`s data

Mailbox

  • Collect all whisper messages
  • Requesting by bloom filter
  • Sending messages directly
